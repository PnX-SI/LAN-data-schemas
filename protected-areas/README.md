## Protected areas data schema

The protected areas data schema allows to harmonise the way basic information about protected areas is published and shared.
It is based on LiveAlpsNature project workshops with European partners and an analysis of existing data and systems (WDPA, OSM, Wikidata, Digitize the planet).
